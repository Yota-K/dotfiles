## 管理しているファイル
* .agignore
* .bash_profile
* .bashrc
* .tigrc
* .tmux.conf
* .vimrc
* iTerm2のテーマ
* Brewfile（Homebrewでインストールするライブラリを管理できるファイル）
* config.fish
