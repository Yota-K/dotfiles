# dotfiles
dotfilesを管理するリポジトリ。

## 管理しているファイル
* .agignore
* .bash_profile
* .bashrc
* .tigrc
* .tmux.conf
* .vimrc
* iTerm2のテーマ
* Brewfile（Homebrewでインストールするライブラリを管理できるファイル）

## 使っているCLIツールなど
* [tmux](https://github.com/tmux/tmux)
* [ag](https://github.com/ggreer/the_silver_searcher)
* [ranger](https://github.com/ranger/ranger)
* [tig](https://github.com/jonas/tig)
